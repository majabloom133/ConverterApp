# Converter App - Workshop

A console-based Java application that converts various units.
Developed as part of the Lexicon Java course.

## Features
- Currency (SEK to USD/EUR)
- Temperature, Length, Weight, Speed, Time
- Power Converter (Demonstrating Static vs Instance members)
- Date and Time timestamps for every conversion
- Data validation using try-catch blocks

## How to run
Run the `Main` class in the `se.lexicon` package.
